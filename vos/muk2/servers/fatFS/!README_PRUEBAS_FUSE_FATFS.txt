2) COPIA DE DATOS DE PC A NETBOOK
	editar el archivo /etc/network/interfaces 
	    cambiar la direccion IP de cada nodo (NODO0 tiene asignada 192.168.1.100)
		
3)	rearrancar networking
		/etc/init.d/networking restart
	configurar interface lo (FUNDAMENTAL PARA SPREAD)
		ifconfig lo 127.0.0.1 netmask 255.0.0.0
	verificar con 
		ifconfig
	
		
4)	Transferir de la PC al MoL_Module\dvd-scripts /etc/fstab
	Este contiene la linea agregada:
		none            /dev/pts        devpts        gid=5,mode=620    0 0
	Luego montar para poder hacer SSH  
		mount /dev/pts
